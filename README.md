　「Custom System Builder」と「BCDice for Foundry VTT」を利用した汎用的に使えるワールドセット

## インストール

FVTTの「ワールド」の「ワールドを入手」ウィンドウで、「URLを指定」の欄に「ManifestURL」をペーストしてインストールしてください。ManifestURLはコアのバージョンに対応するものを選択してください。

複数のモジュールのインストールが促されます。必須モジュールは導入してください。推奨モジュールの導入は任意です。

- v11: https://github.com/jeannjeann/csb-universal-fvtt/releases/download/v11/world.json
- v12: https://github.com/jeannjeann/csb-universal-fvtt/releases/download/v12/world.json
- v13: https://github.com/jeannjeann/csb-universal-fvtt/releases/download/v13/world.json
